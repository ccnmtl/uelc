Management Panel

Here you can manage cases, cohorts and users.

To get started:

Step 1: Create a cohort. Cohorts experiences with one or more users and may include one or more cases.

Step 2: Create a case or edit an existing case and associate it with a cohort. Cases are a collection of pages that include text, decision and gate blocks. Each case is a new decision experience for a cohort of users.

Step 3: Create users or add existing users to a cohort. Users have three profile types.

Administrators may access the ELDEx management panel, facilitate case control, edit pages, and view content from the perspective of other profile types.

Assistants may facilitate case control, edit pages and view content from the perspective of other profile types.

Group users are given to group leaders to register group activity in the system. Each group profile is given a case path in the case control for the assistant and admin users to facilitate.
